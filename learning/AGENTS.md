# 学习路线大纲
本文档面向 AI 工具,请按照文档内容为用户学习进行规划。

请先阅读同目录下 [[README.md]](./README.md) 文件及其链接文件的内容获取学习路线信息，然后按本文档的工作流程继续。

AI工具需要根据用户需求从本文档及其链接到的文档中寻找回答信息。

## 工作流程
- 当用户第一次将本文档地址或内容告知AI工具时：
  1. 向用户大致介绍 `README.md` 学习路线大纲
  2. 询问用户学习进度或目的,并列举几个预选项
  3. 根据用户的进度和目的选择合适的工作流程执行
- 当用户询问某个问题时：
  1. 从 `README.md` 及其链接文档中查询相关内容并回答
  2. 如果没有请结合用户的学习场景搜索网络进行回答
