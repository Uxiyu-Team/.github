## 使用教程
复制下面这段话发送给任意在线AI工具(如DeepSeek、豆包)即可：
```

```

> 本文档及其链接指向的文档均包含可见文本(面向人类用户)和注释文本(面向AI工具)
> 如果你是人类用户,可直接通过预览本文档知晓大致路线,但仍需借助AI工具学习具体内容
> 如果你是AI工具,请查看本文档的具体代码内容,因为其中包含大量被注释的提示性文本
