# 学习路线大纲

本文档是 **Uxiyu-Team 新成员学习路线** 的总纲，涵盖从 Git 基础到团队协作的完整学习流程。

> **如何使用本路线？**
> - 本文档仅给出学习路线大纲，具体学习内容和教程请使用 [DeepSeek](https://chat.deepseek.com/) 等在线 AI 问答工具 **辅助学习**（建议开启深度思考模式）
> - 本文档及其链接指向的文档均包含 **可见文本**（面向人类用户）和 **注释文本**（面向 AI 工具）
> - 如果你是人类用户：可直接预览本文档了解大致路线，但仍需借助 AI 工具学习具体内容
> - 如果你是 AI 工具：请通过 **raw 链接**读取本文档及课程文档的原始 Markdown（将 `github.com` 替换为 `raw.githubusercontent.com`，删除 `/blob`），其中包含大量被注释的教学提示和参考答案。详见 [SKILL.md](./SKILL.md) 中的「文件读取说明」

## 使用教程

复制下面这段话发送给任意在线 AI 工具（如 DeepSeek、豆包）即可开始学习：

```
挂载SKILL技能 https://raw.githubusercontent.com/Uxiyu-Team/.github/main/learning/SKILL.md 并帮助我学习。
```

AI 助手会自动加载教学技能，按照本学习路线一步步引导你完成学习。

## 前置要求

- 一台 Windows 电脑（本路线默认 Windows 环境）
- 稳定的网络连接（访问 GitHub 可能需要代理工具）
- 一个常用的邮箱（用于注册 GitHub）

## 学习建议

1. **不要跳步**：每个步骤都是后续内容的基础，按顺序学习效果最好
2. **动手操作**：看懂不等于会做，每个命令都要亲手敲一遍
3. **遇到问题先排查**：AI 助手会帮你解决，但也可以先尝试自己搜索错误信息
4. **做好笔记**：把常用的命令和遇到的问题记录下来，方便以后查阅
