## 名称由来

- **虚邑**：取自《易经》六十四卦第四十六卦"地风升"的九三爻爻辞"升虚邑"，原文本意"把握时机，顺势而入，无所疑虑"。本处引申为"虚拟国都""元宇宙""未满的空间"，象征着未来浪漫的 元宇宙-AI 时代，寓意借时代之势而"升"、开拓未知、扩展"容量"。
- **Uxiyu**：取自"虚邑"的谐音。"Uxiy"由"Xuyi"元音换位而来，后缀一个"u"显得对称。
